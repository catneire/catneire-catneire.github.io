Clasificación de los lenguajes de marcas

Tipos y clasificación de los más relevantes

Tipo Descripción Ejemplo
Marcado de presentación Indica el formato del texto, es decir cómo ha de presentarse el documento. Las etiquetas de marcado suelen estar ocultas al usuario. Este tipo de lenguajes son los usados tradicionalmente en procesadores de texto y codifican como ha de presentarse el documento. Por ejemplo, indicando que una determinada palabra debe presentarse en fuente itálica. Microsoft Word.
Marcado de Procedimiento Enfocado hacia la presentación del texto, sin embargo, las marcas que formatean el texto son visibles para el usuario y permiten procesamiento (realizar un conjunto de acciones) según el tipo de etiqueta. Las etiquetas se integran dentro de un marco que controla qué porciones de código se ejecuta. LaTeX, HTML
Marcado descriptivo o semántico Utiliza las marcas o etiquetas para describir los fragmentos de texto. Este tipo de lenguajes no definen qué se debe hacer con una sección del documento si no que las marcas sirven para indicar qué es esa información. XML
